VS Code Setup

Install the Raspberry Pi Pico Extension

The official Raspberry Pi Pico extension for Visual Studio Code simplifies project creation, development, and debugging for Pico boards. It includes features like a project generator, automatic toolchain management, and one-click compilation. :contentReference[oaicite:1]{index=1}

Pulling

To clone the repository and its submodules, execute the following commands:

git clone https://github.com/CircuitReeRUG/macro_pad.git
cd macro_pad
git submodule update --init --recursive
